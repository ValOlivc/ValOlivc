### 📂 Mesa de Partes Digital — Colegio San Miguel (Piura)

Sistema web para la gestión digital de documentos administrativos en una institución educativa.

**Características principales:**
- 🔐 Registro e inicio de sesión de usuarios
- 📨 Envío y seguimiento de documentos digitales
- 🗂️ Panel administrativo para gestión de solicitudes
- 📅 Organización por estado y fecha
- 📱 Interfaz web responsive
